Privacy First

Privacy policy built for a debugging tool.

UAF Debugger Pro is designed to inspect ad-related elements on the active page and help developers enable a temporary debug cookie. It is not built to collect personal information, track users, or send browsing data to external servers.

At a glance

  • No personal data collection The extension does not collect, store, or sell personal information.
  • Local browser actions only It reads the active page to find supported ad units and banner wrappers.
  • Cookie control for debugging It can create or remove the `uadrefresh_debug` cookie on the current site.

Information We Access

The extension only accesses information necessary to operate its debugging features on the active browser tab. This includes page elements related to ad units, banner wrappers, and the `uadrefresh_debug` cookie used for testing UAF behavior.

Active tab URL context Ad unit identifiers Banner wrapper IDs Debug cookie state

How This Information Is Used

  • To detect supported ad units and banner wrappers on the current page.
  • To let the user enable or disable the `uadrefresh_debug` cookie for debugging purposes.
  • To scroll to and visually highlight selected elements when the user clicks them in the extension popup.
  • To display the expiration time of the debug cookie in the extension interface.

Data Sharing

UAF Debugger Pro does not transmit page data, cookie data, or user information to external services. No analytics, remote tracking scripts, or third-party data brokers are used by this extension.

Storage and Security

Any small amount of state used by the extension, such as cookie expiration metadata, stays inside the browser through Chrome extension storage. The tool is intended to support internal debugging workflows and does not maintain a remote database of user activity.

Policy Updates

If the extension behavior changes in a way that affects privacy, this page should be updated to reflect the new functionality before release. Users should review this policy periodically for the latest version.
